## Установка и запуск
1. Клонируйте репозиторий:
```sh
git clone https://github.com/AngelsAndDemonsDM/DM-Bot.git
```

2. Перейдите в директорию проекта:
```sh
cd DM-Bot
```

3. Установите зависимости:
```sh
pip install -r requirements.txt
```

4. Запустите:
```sh
python3 Code.DM-Bot/main.py
```
