POST /account/logout

Данный endpoint служит для выхода пользователя из системы. Пользователь должен предоставить свой токен для завершения сеанса.

Заголовки:

token (тип: string, обязательный: да) - Токен пользователя, который пытается выйти из системы.

Возвращаемые данные:

202 Accepted - JSON объект с сообщением о том, что попытка выхода была выполнена:

{
    "message": "Logout attempt was made"
}

403 Forbidden - JSON объект с ошибкой, если доступ запрещен:

{
    "message": "Access denied"
}

500 Internal Server Error - JSON объект с сообщением об ошибке на сервере:

{
    "message": "An unexpected error occurred",
    "error": "Detailed error message"
}
