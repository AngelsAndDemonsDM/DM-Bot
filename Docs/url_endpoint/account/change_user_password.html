POST /account/change_user_password

Данный endpoint служит для изменения пароля пользователя. Запрашивающий пользователь должен иметь флаг change_password или быть самим пользователем, чей пароль изменяется.

Может вернуть все стандартные коды ответов.

Заголовки:

user_token тип: string, обязательный - Токен пользователя, запрашивающего изменение пароля.

Параметры:

login тип: string, обязательный - Логин пользователя, чей пароль будет изменен.
new_password тип: string, обязательный - Новый пароль для пользователя.

Возвращаемые данные:

200 OK - JSON объект с сообщением об успешном изменении пароля:

{
    "message": "Password changed successfully"
}
