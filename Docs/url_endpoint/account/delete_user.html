POST /account/delete_user

Данный endpoint служит для удаления пользователя. Запрашивающий пользователь должен иметь флаг delete_users.

Может вернуть все стандартные коды ответов.

Заголовки:

user_token (тип: string, обязательный: да) - Токен пользователя, запрашивающего удаление пользователя.

Параметры:

login (тип: string, обязательный: да) - Логин пользователя, которого нужно удалить.

Возвращаемые данные:

200 OK - JSON объект с сообщением об успешном удалении пользователя:

{
    "message": "User deleted successfully"
}
