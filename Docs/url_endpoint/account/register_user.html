POST /account/register

Данный endpoint служит для регистрации нового пользователя. Пользователь должен предоставить свой логин и пароль для создания учетной записи.

Из стандартных коды ответов может вернуть только 400 и 500

Параметры:

login (тип: string, обязательный: да) - Логин пользователя.
password (тип: string, обязательный: да) - Пароль пользователя.

Возвращаемые данные:

200 OK - JSON объект с сообщением об успешной регистрации и токеном доступа:

{
    "message": "Register successful",
    "token": "your_token_here"
}
