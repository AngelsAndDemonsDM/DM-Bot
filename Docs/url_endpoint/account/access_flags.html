POST /account/access_flags

Данный endpoint служит для получения флагов доступа пользователя. Требует предоставления токена пользователя в заголовках запроса.

Может вернуть все стандартные коды ответов.

Заголовки запроса:

  • Authorization: Bearer {requester_token}

Возвращаемые данные:

200 OK - JSON объект с флагами доступа пользователя:

{
    "flag1": "value1",
    "flag2": "value2",
    ...
}
