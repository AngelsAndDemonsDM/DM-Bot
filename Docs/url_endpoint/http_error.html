Базовые возможные ответы от любых запросов

Данная страница описывает базовые ответы от endpoint'тов.

Возможные ответы

Код 400 (Bad Request)

{
    "message": "Field(s) <названия полей> are required"
}

Этот код возвращается, когда в запросе отсутствуют необходимые поля.

Код 401 (Unauthorized)

{
    "message": "Unauthorized"
}

Этот код возвращается, если пользователь не авторизован.

Код 403 (Forbidden)

{
    "message": "Access denied"
}

Этот код возвращается, если доступ к ресурсу запрещен.

Код 500 (Internal Server Error)

{
    "message": "An unexpected error occurred",
    "error": "<описание ошибки>"
}

Этот код возвращается при непредвиденной ошибке на сервере.
