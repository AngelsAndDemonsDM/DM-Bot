URL endpoint status

GET /status

Данный endpoint служит для проверки статуса сервиса. Возвращает сообщение о том, что сервис работает.

Параметры:

Данный запрос не требует параметров.

Возвращаемые данные:

200 OK - JSON объект с сообщением об успешной проверке статуса:

{
    "message": "Service is running"
}
