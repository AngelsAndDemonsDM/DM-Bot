POST /server/download

Данный endpoint служит для загрузки архива с текстурами. Пользователь должен предоставить свой токен для получения доступа к архиву.

Из стандартных коды ответов может вернуть только 401 и 500

Заголовки:

user_token тип: string, обязательный - Токен пользователя, запрашивающего загрузку архива.

Возвращаемые данные:

200 OK - Архив с текстурами в формате ZIP:

{
    "message": "Archive downloaded successfully",
    "file": "sprites.zip"
}
