POST /server/access_flags

Данный endpoint служит для получения флагов доступа пользователя. Требует предоставления токена пользователя в заголовках запроса.

Заголовки запроса:

  • Authorization: Bearer {requester_token}

Возвращаемые данные:

200 OK - JSON объект с флагами доступа пользователя:

{
    "flag1": "value1",
    "flag2": "value2",
    ...
}

403 Forbidden - JSON объект с сообщением об отказе в доступе:

{
    "message": "Access denied"
}

500 Internal Server Error - JSON объект с сообщением об ошибке:

{
    "message": "An unexpected error occurred",
    "error": "Error details"
}
